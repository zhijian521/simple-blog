搭建一个属于自己的网站，是作为一个开发者探索互联网世界、践行开源与分享精神的有趣起点。

这篇文档记录了我使用腾讯云轻量应用服务器与宝塔面板搭建个人网站的完整流程。

选择这个方案，主要是考虑到腾讯云在域名与服务器统一管理上的便利性，且轻量应用服务器的价格（特别是活动期间）对个人开发者非常友好。

以下是搭建过程中整理的实际步骤与配置细节，希望能为有类似需求的朋友提供一份参考。

---

## 1. 域名准备

域名是个人网站在互联网上的标识。

- **域名选购**：访问 [腾讯云域名注册](https://buy.cloud.tencent.com/domain)，搜索并选择心仪的域名。
- **实名认证**：根据合规要求，新注册域名需要关联实名认证信息模板。如果没有现成模板，可以先在后台创建一个。
- **后缀选择**：
  - `.cn`：国内解析速度和性价比优秀，推荐作为首选；
  - `.com`：国际通用，认知度最高，价格稍贵；
  - `.top` / `.xyz`：适合预算有限的实验性项目。

## 2. 服务器配置

对于个人博客或小型应用，轻量应用服务器（Lighthouse）是一个性价比较高的选择。

- **配置建议**：2 核 CPU、2 GB 内存、4 Mbps 带宽，系统镜像选择「宝塔 Linux 面板」，这样可以省去手动配置 Web 面板的步骤。
- **记录公网 IP**：购买完成后，在控制台的服务器列表中获取公网 IP。这个 IP 将用于后续的域名解析与服务连接。

> 路径：控制台 → 轻量应用服务器 → 服务器列表 → 查看 IP

## 3. ICP 备案实践

如果服务器选在境内机房，根据相关政策，域名必须通过 ICP 备案才能开通 80/443 端口的 Web 访问。

- **备案入口**：在腾讯云控制台搜索“备案管理”，按照指引填写主体信息与网站信息，并使用微信小程序进行人脸核验。
  > 路径：控制台 → 备案管理 → 开始备案
- **经验分享**：
  - **前置条件**：备案需要绑定已购买的境内服务器，因此需要先购买服务器再发起备案申请。
  - **时间周期**：各省管局的审核时间不同，通常在 7 至 20 个工作日左右。建议在规划建站时预留足够的时间。
  - **要求挂载**：备案通过后，需在网页底部添加备案号并链接至工信部网站。
  - **境外备选**：如果对免备案有硬性需求，可以考虑选择中国香港或境外节点的服务器，但访问延迟和稳定性可能需要权衡。

## 4. 域名解析

备案通过后，即可配置域名解析，将域名流量导向服务器。

> 路径：控制台 → 我的资源 → 域名注册 → 选择域名 → 解析

在解析控制台添加以下两条 `A` 记录：
- **主机记录 `@`**：记录类型 `A`，记录值填写服务器公网 IP；
- **主机记录 `www`**：记录类型 `A`，记录值同样填写服务器公网 IP。

## 5. 配置服务器环境

为了让面板能够正常访问，并搭建好运行环境，我们需要进行以下配置。

### 5.1 开放安全组端口
宝塔面板默认使用 `8888` 端口，需在腾讯云控制台的防火墙中添加放行规则。
> 路径：轻量应用服务器 → 防火墙 → 添加规则（放行 TCP 8888 端口）

### 5.2 获取面板初始凭证
在控制台登录到服务器，执行以下命令获取宝塔面板的默认登录地址、账号与密码：
```bash
sudo /etc/init.d/bt default
```

### 5.3 登录并部署环境
在浏览器中访问输出的地址，登录后可根据个人项目的具体技术栈选择一键安装：
- **Web 服务器**：推荐 Nginx（轻量、高性能）；
- **数据库**：MySQL（根据个人需要，静态网站可不装）；
- **脚本语言**：PHP（或根据需求部署 Node.js/Go 环境）。

建议登录后绑定宝塔账号，以便后续利用面板提供的免费 SSL 证书申请与数据自动备份工具。

## 6. 部署与上线

有了运行环境，便可以部署实际的项目文件。

### 6.1 添加站点
> 路径：宝塔面板 → 网站 → 添加站点
在弹出的窗口中输入你的域名，其他选项保持默认即可。

### 6.2 上传代码
创建站点后，点击对应的根目录路径进入文件管理器。只需将文件上传至该目录并解压，即可通过域名直接访问。

## 7. 安全配置建议

在享受建站乐趣的同时，保障服务器的安全性同样重要。以下是搭建过程中建议完成的几项安全初始化操作：

- **更改默认端口**：修改宝塔面板的 `8888` 默认端口（修改后需同步在腾讯云防火墙中放行新端口，并关闭原 8888 端口）；
- **修改默认账号与密码**：避免使用系统生成的弱口令或初始默认账号；
- **配置面板安全入口**：自定义登录 URL 路径，防止被扫描器恶意爆破。

> 路径：宝塔面板 → 面板设置 → 安全设置

## 常见问题与排查经验

在实践中，遇到问题是很好的学习契机。以下是几个常见的排查思路：

1. **域名解析未生效**：
   - 检查备案状态是否已同步，未备案的域名在境内服务器上会被系统自动拦截。
   - DNS 刷新需要时间，通常在 10 分钟到数小时内生效，可以使用 `ping 你的域名` 命令检查解析是否指向了正确的 IP。
2. **面板或网站无法访问**：
   - 检查腾讯云控制台的「防火墙」与宝塔面板内部的「系统安全」是否双重放行了对应端口（如 80、443、8888）。

## 后续与持续探索

个人网站的搭建只是第一步，互联网的魅力在于持续的维护与内容创作。在后续的运维中，建议大家：
- **制定备份策略**：利用宝塔的计划任务，定期将网站数据与数据库备份至对象存储或本地，防范数据丢失风险。
- **全面启用 HTTPS**：通过面板内置的 Let's Encrypt 免费申请 SSL 证书，保护数据传输安全。
- **探索开源生态**：尝试将代码托管至 GitHub，或通过 CI/CD 自动化部署，体验更现代化的开发工作流。

搭建网站的过程也是对网络协议、系统运维等基础知识的实践。

祝大家的建站之旅顺利，期待在开源的世界里看到更多有趣的个人站点。
