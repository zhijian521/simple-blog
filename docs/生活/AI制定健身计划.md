AI 发展很快，不仅仅可以用来写代码，还可以做一些其他有意思的事情。比如： AI 可以根据你的个人情况，为你量身定制健身计划。

## 为什么用 AI 制定健身计划

- **个性化**：根据年龄、身高、体重、目标等定制
- **科学性**：基于运动科学和营养学原理
- **灵活性**：可随时调整，适应生活节奏
- **低成本**：无需私人教练，零门槛使用

## 核心提示词模板

将以下提示词复制到 AI 工具（如 ChatGPT、Claude、Kimi 等），填写你的个人信息：

```
你是一位专业的健身教练和营养师，拥有丰富的运动科学知识。请根据我的个人情况，为我制定一份科学、可执行的健身计划。

## 我的个人信息

- **年龄**：[填写年龄] 岁
- **性别**：[男/女]
- **身高**：[填写身高] cm
- **体重**：[填写体重] kg
- **目标**：[减脂/增肌/塑形/提升体能/其他]
- **目标体重**：[填写目标体重] kg（可选）
- **运动经验**：[无/新手/中级/高级]
- **每周可锻炼次数**：[填写次数] 次
- **每次可锻炼时长**：[填写时长] 分钟
- **可用的健身设备**：[无/哑铃/杠铃/健身房/其他]
- **身体状况**：[无特殊情况/膝盖问题/腰背问题/其他]
- **饮食习惯**：[正常/素食/低碳/高蛋白/其他]
- **作息时间**：[早睡早起/晚睡晚起/作息不规律]
- **偏好运动类型**：[有氧/力量/瑜伽/游泳/跑步/无偏好]

## 请提供以下内容

1. **身体分析**
   - BMI 指数及健康状况评估
   - 基础代谢率（BMR）计算
   - 每日推荐热量摄入
   - 目标可行性分析

2. **饮食建议**
   - 每日三大营养素分配（碳水/蛋白质/脂肪）
   - 一周饮食计划示例
   - 推荐食物清单
   - 需要避免的食物
   - 进食时间安排

3. **训练计划**
   - 每周训练日程安排
   - 每次训练的具体内容（动作、组数、次数、休息时间）
   - 有氧与力量训练的比例
   - 训练强度建议（心率区间、RPE）

4. **注意事项**
   - 常见错误及避免方法
   - 受伤预防建议
   - 训练进阶策略
   - 休息与恢复建议

5. **进度追踪**
   - 如何记录训练数据
   - 如何测量身体变化
   - 多久调整一次计划
   - 遇到平台期怎么办

请用通俗易懂的语言解释专业术语，确保我可以轻松理解并执行。
```


## 实际案例

```
你是一位专业的健身教练和营养师，拥有丰富的运动科学知识。请根据我的个人情况，为我制定一份科学、可执行的健身计划。

## 我的个人信息
- **年龄**：28岁
- **性别**：男
- **身高**：170cm
- **体重**：69kg
- **目标**：减脂/增肌/塑形/提升体能
- **目标体重**：62.5kg
- **运动经验**：中级
- **每周可锻炼次数**：5 次
- **每次可锻炼时长**：30 分钟
- **可用的健身设备**：哑铃/自行车/弹力带
- **身体状况**：无特殊情况
- **饮食习惯**：正常
- **作息时间**：早睡早起
- **偏好运动类型**：有氧/力量/瑜伽/
...more...

```

输出内容为（以下内容为 GPT-5.4 输出）：

```
• 以下方案按“减脂兼顾增肌”的思路设计。你现在并不胖，最适合你的不是猛减体重，而是“小幅热量缺口 + 规律力量训练 + 适量有氧”，也就是常说的“体脂重组”。下面的热量和营养分配，是按公式和你的训练频率推算的估算值，不是体成分仪实测值，先执行 2 周，再按体重和腰围微调。

1. 身体分析

- BMI = 69 ÷ 1.70² = 23.9。按 CDC 成人标准，属于健康范围。
- BMR = 10×69 + 6.25×170 - 5×28 + 5 = 1618 kcal/天。BMR 的意思是“你躺着不动一天，身体也会消耗的热量”，它不是你的吃饭目标。
- 按你每周 5 次、每次 30 分钟训练估算，你的维持热量大约在 2300-2450 kcal/天。
- 你的减脂塑形起始摄入建议：2050-2150 kcal/天。先从 2100 kcal 左右开始最稳。
- 目标体重 62.5kg 对应 BMI 约 21.6，可行，但不建议追太快。
- 更现实的节奏：每周下降 0.25-0.5kg。按这个速度，预计需要 4-6个月。
- 阶段目标更合理：
   - 第 1 阶段 8 周：到 66-67kg，腰围下降 3-5cm
   - 第 2 阶段 8-12 周：到 64-65kg
   - 到这一步再看体型，未必一定要硬减到 62.5kg

2. 饮食建议

- 先按 2100 kcal 执行。
- 蛋白质：140g/天 左右。作用是保肌肉、增加饱腹感。
- 碳水：230-250g/天。作用是给训练供能，不用妖魔化。
- 脂肪：55-60g/天。作用是维持激素、关节和恢复。
- 大致占比：碳水 45%-48%，蛋白质 26%-28%，脂肪 24%-26%。
- 纤维建议 25-35g/天，饮水 2.5-3L/天。
- 进食时间安排：
   - 起床后 1-2小时 内吃早餐。
   - 每天把蛋白质分到 3-4餐，每餐 30-40g 左右。
   - 训练前 60-120分钟 吃一顿正常餐；如果赶时间，训练前 30-60分钟 吃香蕉 + 酸奶/乳清也可以。
   - 训练后不用迷信“30 分钟黄金窗口”，但最好在 1-2小时 内补到 25-40g蛋白 + 30-60g碳水。
   - 晚餐尽量放在睡前 2-3小时。
   - 如果你早上训练，训练前吃轻一点；如果你晚上训练，午餐和训练前加餐的碳水要够。
- 一周饮食示例：
   - 周一：燕麦 + 牛奶 + 鸡蛋 + 香蕉；米饭 + 鸡胸 + 西兰花；三文鱼 + 红薯 + 时蔬；加餐无糖酸奶。
   - 周二：无糖酸奶 + 燕麦 + 苹果；杂粮饭 + 牛肉 + 青菜；虾仁豆腐 + 南瓜 + 菌菇；加餐乳清蛋白。
   - 周三：全麦面包 + 鸡蛋 + 豆浆；土豆 + 去皮鸡腿 + 蔬菜；米饭 + 瘦牛肉 + 炒时蔬；加餐毛豆。
   - 周四：燕麦粥 + 鸡蛋 + 水果；全麦卷饼 + 金枪鱼/鸡胸 + 生菜；鳕鱼 + 米饭/藜麦 + 西兰花；加餐低糖酸奶。
   - 周五：酸奶碗 + 燕麦 + 蓝莓；米饭 + 牛腱/鸡胸 + 两份蔬菜；虾仁 + 豆腐 + 红薯；加餐乳清 + 苹果。
   - 周六：杂粮粥 + 鸡蛋 + 无糖酸奶；外食时选米饭 + 双份蔬菜 + 清淡蛋白；全麦面 + 鸡胸/牛肉 + 番茄蔬菜；加餐高蛋白牛奶。
   - 周日：豆浆 + 全麦面包 + 煎蛋 + 水果；三文鱼/牛肉饭 + 蔬菜；清汤火锅选瘦肉、虾、豆腐、蔬菜，主食适量；加餐酸奶。
- 推荐食物：
   - 蛋白质：鸡胸、去皮鸡腿、瘦牛肉、鱼虾、鸡蛋、无糖酸奶、牛奶、豆腐、豆浆、乳清蛋白。
   - 碳水：米饭、燕麦、土豆、红薯、玉米、全麦面、杂粮饭、水果。
   - 脂肪：坚果、橄榄油、牛油果、深海鱼。
   - 蔬菜：绿叶菜、西兰花、番茄、胡萝卜、菌菇、黄瓜、彩椒。
- 尽量少吃：
   - 奶茶、含糖饮料、果汁、酒。
   - 油炸食品、蛋糕、饼干、薯片、糖果。
   - 香肠、培根、炸鸡、重油重盐外卖。
   - “看着不多但热量很高”的酱料和零食。
- 如果你不想算卡路里，直接用“餐盘法”：
   - 每餐 1.5掌 蛋白质
   - 1拳 主食，训练后可到 1.5拳
   - 2拳 蔬菜
   - 1拇指 脂肪

3. 训练计划
先说原则：每次 30分钟，固定结构是 热身 4-5 分钟 + 主训练 20-22 分钟 + 放松 3-5 分钟。为了在短时间内提高效率，我会用“超组”，就是两个动作连着做，再休息。

- 强度怎么理解：
   - RPE 是主观用力程度，10分 = 拼尽全力
   - RPE 7 = 还能再做 3 次
   - RPE 8 = 还能再做 2 次
   - Zone 2 有氧 = 还能说短句，但已经明显在喘

1. 周一：力量 A
   杯式深蹲 3×8-10
   哑铃地板卧推 3×8-12
   这两个动作连做，休息 45-60秒，共 3轮
   单臂哑铃划船 3×10-12/侧
   哑铃罗马尼亚硬拉 3×10-12
   这两个动作连做，休息 45-60秒，共 3轮
   平板支撑 2×40-60秒
2. 周二：自行车间歇
   热身骑行 5分钟
   快骑 40秒 + 慢骑 80秒，做 8轮
   快骑强度 RPE 8-9，如果看心率，目标大约 145-160 bpm
   慢骑 RPE 2-3
   最后放松骑 5分钟
   再做大腿前侧、臀部、髋部拉伸 4分钟
3. 周三：力量 B
   反向箭步蹲 3×8-10/侧
   哑铃肩推 3×8-10
   这两个动作连做，休息 45-60秒，共 3轮
   弹力带下拉或划船 3×12-15
   哑铃臀桥/臀推 3×12-15
   这两个动作连做，休息 45-60秒，共 3轮
   死虫式 2×10/侧
4. 周四：恢复有氧 + 瑜伽
   自行车 20分钟 Zone 2，心率大约 115-134 bpm，强度 RPE 5-6
   瑜伽/拉伸 10分钟：猫牛式、下犬式、髋屈肌拉伸、梨状肌拉伸、胸椎打开
5. 周五：力量 C 代谢循环
   做 3-4轮 循环
   哑铃深蹲推举 10次
   俯卧撑 8-12次
   哑铃罗马尼亚硬拉 12次
   弹力带划船 15次
   登山跑 30秒
   动作间休 15-20秒
   每轮后休 60-75秒
   如果还有时间，做侧桥 2×30-40秒/侧

- 周末安排：
   - 至少 1天 完全休息
   - 另 1天 轻松步行 30-45分钟 或者分成饭后散步
   - 你的正式训练已经是每周 150分钟，很不错，但如果想让减脂更顺，日常步数尽量做到 7000-10000步
- 有氧与力量比例：
   - 正式训练按刺激算，大约 力量 60% : 有氧/恢复 40%
   - 你的主线仍然是力量训练，因为塑形和保肌肉更依赖它

4. 注意事项

- 最常见错误是“吃太少”。你本来 BMI 就正常，热量压得太狠，掉的往往先是肌肉和状态。
- 第二个常见错误是“只看体重”。你真正要看的是腰围、镜子、衣服松紧和训练表现。
- 力量动作宁可慢一点、稳一点，也不要靠惯性甩起来。
- 每次训练前都热身；每个新动作第一组先用轻重量找感觉。
- 有酸胀感是正常训练反应，但“锐痛、麻、头晕、胸闷”不正常，出现就停。
- 每周进阶一点点就够了，这叫“渐进超负荷”，意思是比上周多一点 reps、重一点点，或者休息更短一点。
- 进阶规则很简单：某个动作如果所有组都打到目标次数上限，且 RPE 还不超过 7，下周就加一点重量，或者每组多做 1-2次。
- 如果家里哑铃不够重，就用这三招增加难度：放慢下放 3秒、底部停 1秒、改成单侧动作。
- 每连续训练 3周，第 4周 可以主动减量：总组数减少 30%-40%，让身体恢复一下。
- 睡眠尽量保证 7.5-8.5小时。你早睡早起，这是很大的优势。
- 休息日蛋白质不要降，恢复比“少吃一点”更重要。

5. 进度追踪

- 每次训练都记 4 个数据：动作、重量、次数、RPE。
- 体重建议每周测 3-4次，都放在早起上厕所后、空腹、同一时间，再看“周平均值”。
- 腰围每 2周 测一次，位置固定在肚脐水平。
- 每 4周 拍一次正面、侧面、背面照片，光线和站姿尽量一致。
- 计划调整频率：2-4周 看一次，不要天天改。
- 如果连续 2-3周 体重和腰围都没变化，先检查执行率是不是到 85% 以上。
- 确认执行没问题后，再二选一：
   - 每天减少 100-150 kcal
   - 每天多走 1500-2000步，或者饭后多加两次 10分钟 快走
- 如果平台期同时伴随疲劳大、睡眠差、力量下降，就不要继续硬减，先做 5-7天 维持热量，让状态回来。
- 如果力量连续 2周 明显下滑，通常说明热量太低、恢复不够，或者 HIIT 做多了。
```

## 注意事项

1. **循序渐进**：不要一开始就追求高强度，给身体适应时间
2. **记录数据**：体重、体围、训练数据都要记录，便于分析
3. **倾听身体**：感到不适及时调整，避免受伤
4. **保持耐心**：健身是长期工程，不要期待速成
5. **定期评估**：每月评估一次，根据进展调整计划
